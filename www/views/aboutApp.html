About Kai Chronicles

Kai Chronicles is a web application to play the Lone Wolf game books (books 1 to 29). It is an adaptation of the original game books to play them on a web browser.

This fork builds on the work of cracrayol, who continued the original Kai Chronicles by tonib after development ended in 2021.

All book content is distributed by Project Aon under the Project Aon License. Books were written by Joe Dever, and translated by Project Aon volunteers.

Frequently Asked Questions

How many books are supported?

Project Aon has published 29 books from the Lone Wolf series in English. This application supports all of them, but game rules must be customized for each section of each book.

Book 2: Where is the Magic Spear?

WARNING: Contains spoilers. Press to see the answer.

There are two ways to get out of the Tarnalin tunnel alive and finish book 2: have the Magic Spear, or have the Animal Kinship discipline. To get the Magic Spear, after crossing the desert and before reaching Port Bax, take the left fork and aid the people in trouble.

Is this the original game?

No. This is an adaptation of Lone Wolf game books for web browsers. We did not write or translate the books. Books were written by Joe Dever, and translated by Project Aon volunteers.

I found a bug

You can report bugs on the GitHub page. Please specify the book number and section number.

Privacy Policy

This web application does not send or store any personal information on any server.

All game data is stored locally in your web browser (IndexedDB / localStorage). Save games, settings, and progress are kept entirely on your device.

No analytics, tracking, or cookies are used.
